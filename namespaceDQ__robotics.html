◆ C4()

Matrix< double, 4, 4 > DQ_robotics::C4 ( )

Returns a constant MatrixXd 4x4 dimensions representing C4, a diagonal negative unit matrix. Given the jacobian matrix J that satisfies 'vec4(dot_x) = J * dot_theta', where dot_x is the time derivative of the translation quaternion and dot_theta is the time derivative of the joint vector, the following relation is established: 'vec4(dot_x) = C4 * J * dot_theta'.

Returns
A constant Eigen::MatrixXd (4,4).
